# Grafiche Costume Vote

Carica qui i tuoi file PNG da Photoshop per personalizzare i layout dell'app.

## Come usare

Salva i tuoi file PNG in questa cartella (`attached_assets/grafiche/`).  
Nell'app, le immagini sono accessibili tramite il percorso `/grafiche/nome-file.png`.

## Struttura consigliata

- `background.png` — sfondo principale
- `logo.png` — logo o titolo dell'evento
- `costume-placeholder.png` — placeholder per costumi senza foto
- `banner-votazione.png` — banner area votazione
- `banner-risultati.png` — banner area risultati

## Formato consigliato

- Formato: **PNG** (con o senza trasparenza)
- Risoluzione: 1080x1920 per sfondi mobile, 1920x1080 per desktop
- Profilo colore: sRGB
